# HotCook: ポークヒレ デミ風（2食分 / KN-HW10G-B想定）

## 材料（2食分）
- 豚ヒレ 350〜400g
- 玉ねぎ 1/2個
- きのこ 1/2パック
- ケチャップ 少量
- ウスターソース 少量
- ラカント（任意：甘み調整）
- 塩・胡椒

## 手順
1. 豚ヒレを食べやすく切る
2. 内鍋に玉ねぎ→きのこ→豚→調味料の順に入れる
3. 煮込み系で加熱
4. 味見してケチャップ/ウスターを微調整

## 食べ方
- ご飯200g + 本品（当日夜 + 翌日昼に流用）
- ブロッコリーを添えると彩りと栄養バランスが整う

## ポイント
- “洋食欲”を満たすが、揚げない・バターなしで低脂質
